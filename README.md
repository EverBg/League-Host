# Discord Bot

This project sets up a simple Discord bot that can run continuously on a hosting service.

## Setup

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env` and add your Discord bot token.
3. Start the bot:
   `npm start`

## Keep it online 24/7

To keep the bot online all the time, host it on a service such as:
- Railway
- Render
- Fly.io
- VPS (Ubuntu)

The bot is already configured to start with `npm start`.
